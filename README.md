# 量化交易收益计算器

这是一个量化交易收益计算器的静态网页版本，根据胜率、平均亏损率、盈亏比、交易次数计算本金变化曲线。

## 功能特性

- 基本参数输入：胜率、平均亏损率、盈亏比、交易次数、初始本金、仓位比例
- 高级选项：三种波动方式（固定值、正态分布、固定区间）
- 资金曲线实时计算和展示
- 统计指标：初始本金、最终本金、总收益率、翻倍倍数、最大回撤、单笔期望收益
- 美观的图表展示（使用 Chart.js）

## 访问方式

访问 GitHub Pages 即可使用：https://anginian.github.io/equity-calculator/

## 使用说明

1. 在左侧输入您的交易参数
2. 点击「计算并绘制」按钮
3. 右侧会显示统计结果和资金曲线图

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chart.js
